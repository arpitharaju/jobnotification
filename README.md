# Job Notification App — Design System (Foundation)

This is the design system foundation for a premium Job Notification App.

It includes:
- Tokenized colors, typography, spacing, and radii
- Global layout structure (Top Bar → Context Header → 70/30 workspace → Proof Footer)
- Core UI components (buttons, cards, inputs, copybox, error + empty states, checklist proof footer)

Preview pages:
- `index.html` (foundation + mixed components)
- `pages/layout.html` (layout structure)
- `pages/components.html` (components overview)
- `pages/states.html` (error + empty states + badges)

If you enable GitHub Pages for this repo (default branch `main`, root folder `/`), these pages will be reachable publicly.
